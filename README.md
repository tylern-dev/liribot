# liribot
Command Line app that displays answers to a user's query.
This app uses these NPM packages:
1. inquirer
2. twitter
3. node-spotify-api
4. request

You can run the app either by:
1. node liri.js **<command    search query**
2. node liri.js
  
Available Commands:
* my-tweets
* spotify-this-song
* movie-this
* do-what-it-says
